# myshop

This is an online shop demo, for the first part only 

To run this project you need to run : npm run dev 

and both the forntend and the backend will be running 

Note: you will need .env at the root directory with the follwing 

NODE_ENV = develoment
PORT = 5000
MONGO_URI = YOUR_MONGO_URI
